What is it?

Leverage web-sms services to provide reduced cost - wide access communication. For communities and NPO's in South Africa

Costs

  • One instance
    • Short Code via BULKsms - $52/mo
    • Digital Ocean Droplet - $10/mo
    • message - $0.02-0.03 per message
  • set up cost of $156

Savings

  • Flat rate messaging - international and local
    • savings as high as $0.05 per message
  • Consolidate mobile plans
    • No expiration date
    • Add and remove devices at leisure

Price Point

  • Capacity dependent - 10 SMS/second
  • Price as subscription + reduced rate per message
  • Scale per member

Scaling

  • Can spin up multiple instances rapidly
    • Multiple nodes for a single shortcode
    • Private shortcode network
    • Expansion costs < Revenue

Market

  • Over 150,000 NPO's operating in south africa
  • 24.9 million~ in poverty
    • 7.3 million with < $1.25 per day

Product

  • Communications package for an organization and its beneficiaries
    • Open access for beneficiaries
    • No-hassle onboarding process for new devices
    • Harness web integration for smart services
  • Flat fee + rated messages

Additional Services

  • Directory
  • Group Messaging
  • Job Board
  • Open Market
  • Voice integration
